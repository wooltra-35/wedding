
# 청첩장 웹사이트 프로젝트 Blueprint

이 문서는 '상모와 유나의 결혼식' 디지털 청첩장 프로젝트의 기능, 디자인, 기술적 사양을 정의하는 청사진입니다.

## 1. 프로젝트 개요

신랑 양상모와 신부 이유나의 결혼을 축하하고 하객들에게 예식 정보를 효과적으로 전달하기 위한 인터랙티브 웹 청첩장입니다. 아름다운 디자인과 함께 하객의 편의를 위한 동적인 기능을 제공하는 것을 목표로 합니다.

## 2. 구현된 기능 및 디자인 (v3.0)

*   **디자인:** 색상, 폰트, 레이아웃 등 전반적인 시각 디자인 구현.
*   **주요 섹션:** 메인, 초대글, 사진 갤러리, 예식일, 오시는 길, 계좌번호 안내, 연락처, 공유 기능.
*   **인터랙티브 기능:** Firebase 연동을 통한 실시간 방명록 및 참석 의사 전달(RSVP) 기능.

## 3. 최종 기능 안정화 (v3.1)

*   **내비게이션:** 카카오내비 SDK를 적용하여 모바일 길안내 기능 안정화 및 목적지를 주차장으로 정확하게 수정.
*   **버그 수정:** UI 전반의 마이너 버그 수정 및 CSS 복원.

## 4. 갤러리 기능 고도화 (v4.0)

*   **바둑판식 그리드 레이아웃:** 기존 캐러셀을 제거하고, 사진을 바둑판 형태로 표시하는 `photo-grid`를 도입했습니다.
*   **라이트박스 슬라이더:** 그리드 사진을 클릭하면 전체 화면으로 이미지를 볼 수 있는 라이트박스 기능을 구현하여, 좌우로 슬라이드하며 모든 사진을 감상할 수 있습니다.

## 5. 텍스트 레이아웃 수정 (v4.1)

*   **문제:** 초대글(`.invitation-section`)의 본문 텍스트가 화면 폭에 따라 부자연스럽게 줄바꿈되는 현상 발생.
*   **해결:** CSS의 `.invitation-section .message` 클래스에 `word-break: keep-all;` 속성을 추가하여 한글 단어 단위로 줄바꿈이 일어나도록 수정했습니다. 이를 통해 가독성을 향상시켰습니다.

## 6. 내비게이션 기능 개선 (v5.0)

*   **문제:** 카카오내비 링크가 부정확한 목적지로 안내하는 문제 발생.
*   **원인:** 하드코딩된 좌표가 실제 목적지(서초과학화예비군훈련장 강동송파)와 일치하지 않음.
*   **해결:** 카카오맵의 장소 검색(keywordSearch) API를 활용하여 ‘서초과학화예비군훈련장 강동송파’ 키워드로 검색된 장소의 좌표를 동적으로 가져오도록 수정했습니다. 이를 통해 항상 가장 정확한 위치 정보로 내비게이션이 실행되도록 보장합니다.

## 7. 파비콘 추가 (v5.1)

*   **요청:** 웹사이트 탭에 표시될 파비콘을 핑크색 하트 모양으로 변경 요청.
*   **구현:** `images/favicon.svg` 경로에 핑크색 하트 아이콘을 추가하고, `index.html`의 `<head>` 태그에 해당 파비콘을 연결하여 웹사이트에 적용했습니다.

## 8. 카카오내비 정확도 개선 (v6.0)

*   **문제:** 키워드 기반 내비게이션 검색이 간헐적으로 부정확한 위치를 안내하는 문제 지속.
*   **해결:** 사용자가 제공한 정확한 주소("서울 서초구 내곡동 31-61")를 카카오의 `addressSearch` API를 통해 좌표로 변환하여 내비게이션 목적지로 설정했습니다. 이를 통해 키워드 검색의 모호성을 제거하고, 내비게이션의 정확도를 대폭 향상시켰습니다. 목적지 이름은 "보넬리가든 (주차장)"으로 명확히 표시하여 사용자의 편의를 더했습니다.

## 9. 신랑/신부 데이터 확인 방법

*   Firebase Console (console.firebase.google.com)에 로그인합니다.
*   해당 프로젝트를 선택합니다.
*   왼쪽 메뉴에서 `빌드` > `Firestore Database`로 이동합니다.
*   `guestbook`과 `rsvps` 컬렉션을 클릭하여 하객들이 남긴 방명록과 참석 여부 응답을 실시간으로 확인할 수 있습니다.
